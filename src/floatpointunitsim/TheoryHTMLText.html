FLOATING POINT ARITHMETIC

Programming languages support other kind of numbers than integers, these numbers are called reals in mathematics. Some examples are

3.14159265... [pi]

2.71828... [e]

An alternative notation commonly used is called  scientific notation and which has a single digit to the left of the decimal point. A number in scientific notation that has no leading zeros is said to be normalized, for example:

3,155,760,000 ten can be represented in scientific notation as 3.15576x10 9 (seconds in a typical century).

In computer arithmetic such numbers are called  floating point numbers because they represent numbers in which the binary point is not fixed unlike the integers.

The programming language C uses the name  float for such numbers and just as in scientific notation, numbers are represented as a single nonzero digit to the left of the point. In binary, the form is

1.xxxxxxxx two x 2 yyyy

Biased notation, with the bias being the number substracted from the normal, unsigned representation to determine the real value.

There exists a compromise between the size of the fraction and the size of the exponent, this tradeoff is between precision and range: increasing the size of the fraction enhances the precision of the fraction, while incresing the size of the exponent increases the range of the numbers that can be represented.
